## Para rodar o jogo:

1. Criar um ambiente virutal:

        python3 -m venv <nome_venv>
2. Ativar o ambiente virtual:

        # Windows
        <nome_venv>\Scripts\activate.bat
        
        #Linux
        source <nome_venv>/bin/activate
3. Instalar pacotes necessários:

        pip install -r requirements.txt

4. Rode o programa:

        python3 connect4.py
